Het onstaan van deze blog

Voor het vak Professional Networking hebben wij, als laatstejaarsstudenten de opdracht gekregen om een blog op te stellen. In deze blog documenteer ik over evenementen die ik heb moeten bijwonen.

Om deze site te maken heb ik gekozen voor simpele HTML en CSS codering. Er waren andere manieren om een site te maken, maar dit is de simpelste manier.

Deze website is gehost door GitHub Pages. Dit was makkelijk om te beslissen, want het is gratis en zeer simpel om op te stellen. De hosting werd geconfigureerd in 5 minuten.

Updates van de blogs gebeuren via GitHub commando's die ik uitvoer via een commandprompt. Eerst zal ik de aanpassingen lokaal uitvoeren en dan zal ik deze pushen op Github, zodat de site bijgewerkt wordt.
